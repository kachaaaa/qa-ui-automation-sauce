# UI Automation Project (Selenium + Pytest)

## 📌 Description
This project contains automated UI tests for the login functionality of SauceDemo application.

## 🛠 Tech Stack
- Python
- Selenium WebDriver
- Pytest

## 📂 Project Structure
pages/ - Page Object classes  
tests/ - Test cases  
conftest.py - pytest fixtures  

## 🚀 How to run tests

Install dependencies:
pip install -r requirements.txt
Run tests:
## 🌐 Tested Application
https://www.saucedemo.com/

## ✅ Covered Test Cases
- Valid login
- Invalid login
- Empty username
- Empty password
- Empty fields
- Locked out user
